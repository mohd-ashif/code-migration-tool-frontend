# Migration Tool Frontend

This is the frontend package for the migration tool application, built with Vite, React, and TypeScript.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Features

- React + TypeScript application
- Vite-powered frontend build
- Tailwind CSS support

## Project structure

- `src/` - application source code
- `src/main.tsx` - entry point
- `src/App.tsx` - main app component
- `src/components/` - UI components
- `src/services/` - API and service logic

## Notes

- The package is private and used as part of a monorepo-style repository.
- Add `.env` files only if needed for local configuration.
